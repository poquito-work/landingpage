### UI/UX Design Direction: Cinematic Luxury Studio Pattern

**Layout Philosophy & Visual Grid**
- Adapt a world-class Game Studio Portfolio structure (Reference: Forge Studio on Behance). The layout must feel editorial, premium, and container-free, using absolute whitespace instead of generic boxes to separate sections.
- Main background canvas must be deep board-green (#143322). Text and key lines must leverage crisp off-white (#f9f2e4) for high-contrast prestige, with Rust (#b65a2f) acting strictly as an intentional, high-end call-to-action signature.
- Typography: Drive the layout using Hero font in ALL CAPS for prominent headers. Track out letters elegantly (letter-spacing) to mimic modern luxury fashion and top-tier AAA gaming studios.

**Interactive Interactions & Gamification Motion**
- **Section 1 (Hero Mosaic):** Implement a flawless $4 \times 4$ or $5 \times 5$ grid on the right. On page load, it displays off-white (#f9f2e4) tile backs. After a 0.5s delay, trigger a smooth, cascading 3D CSS flip wave using `preserve-3d` and a heavy perspective warp. The individual tile segments must seamlessly lock together to form the unified `Poquito_Logo_transparent.png`. Ensure physics-based cursor hover tilting is active post-reveal to give the mosaic organic weight.
- **Section 2 (Playground):** Display the 5 app features as a clean, asymmetrical list. On hover, the text slides slightly to the right while a minimalist, off-white Mahjong suit icon subtly fades into view behind it with a rust accent tint.
- **Section 5 (Interactive FAQ):** Build an acoustic accordian interface. Expanding a question triggers an instantaneous, crisp mechanical expansion accompanied visually by a subtle "clack" bounce animation, mimicking the tactile feedback of two heavy bone tiles colliding.
